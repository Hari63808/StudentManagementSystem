# 🎓 Student Management System

## 📌 Project Overview
The Student Management System is a web-based application developed using Java Servlets and JSP. It allows users to manage student records efficiently by performing operations like adding, viewing, updating, and deleting student details.

---

## 🚀 Features
- Add new student details
- View all students
- Update student information
- Delete student records
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- Java (Servlets)
- JSP (Java Server Pages)
- HTML, CSS
- JDBC
- MySQL Database
- Apache Tomcat Server
